# TaskTracker

Автор: 

## Описание
Учебный проект — консольный трекер задач.

## План функций
- Добавить задачу
- Показать список задач
- Изменить статус задачи
- Удалить задачу

## Стек
C# / .NET 8 / Visual Studio

## Как запустить
1. Открыть solution в Visual Studio
2. Нажать F5
3. В консоли появится меню
