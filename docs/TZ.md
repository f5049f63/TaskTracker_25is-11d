# Мини-ТЗ: TaskTracker

## 1. Название
TaskTracker (Трекер задач)

## 2. Цель
Программа помогает хранить список задач и менять их статус.

## 3. Пользователь
Обычный пользователь ПК (студент).

## 4. Функциональные требования (FR)
- FR1: Добавить задачу (название обязательно)
- FR2: Показать список задач
- FR3: Изменить статус задачи (New / InProgress / Done)
- FR4: Удалить задачу

## 5. Нефункциональные требования (NFR)
- Приложение запускается без ошибок
- Понятные сообщения об ошибках
- Данные будут храниться локально (в будущем — JSON)

## 6. Модель данных

Задача Task:
- Id (int)
- Title (string, не пустая)
- Status (New / InProgress / Done)

## 7. Критерии приемки
- Можно добавить 3 задачи
- Можно изменить статус задачи
- Программа не падает при ошибках ввода

## 8. План модулей
- Core
- App
